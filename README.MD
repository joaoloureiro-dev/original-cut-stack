💈 Barber Shop Website

📌 About the Project

This is a responsive barber shop website built with HTML, CSS and JavaScript.

The goal of this project is to create a modern, mobile-first experience for a real business, focusing on usability and clean design.

---

🚀 Features

- Responsive design (mobile-first)
- Sidebar navigation menu
- WhatsApp direct booking
- Barber showcase section
- Clean and modern UI

---

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

📱 Mobile First Approach

The website was designed primarily for mobile users, ensuring a smooth experience across all devices.

---

📞 Contact

Users can directly book appointments via WhatsApp integration.
